# M-QAM BER Simulation for 4-QAM

This repository contains the MATLAB implementation and project files for a digital communication system based on M-QAM modulation, with emphasis on the BER analysis for 4-QAM.

The project simulates signal transmission through an AWGN channel, compares theoretical and experimental BER results, and applies Support Vector Machine (SVM)-based classification to detect the received constellation symbols.

This work was presented at the 1st Student Conference on Informatics & Telecommunications, University of Peloponnese, in June 2025.
